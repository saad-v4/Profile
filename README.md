### 🚀 نبذة عن المشروع
هذا المستودع يمثل الواجهة الرقمية الخاصة بي، حيث قمت ببناء موقع ويب بسيط واحترافي لتعريف الزوار بمهاراتي البرمجية، وخاصة في مجالات تطوير الويب، البرمجة بلغة **C++** و **C#**، وشغفي في تطوير ألعاب الـ **2D Pixel Art**.

### 🛠 التقنيات المستخدمة
* **HTML5:** لبناء الهيكل التنظيمي للموقع.
* **CSS3:** لتنسيق الألوان، الخطوط، وضمان استجابة الموقع (Responsive Design).
* **Git:** لإدارة وتوثيق مراحل تطوير المشروع.
